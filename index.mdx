# Welcome to ZephyrDB

ZephyrDB is a serverless database built for developers who move fast. With real-time capabilities, global edge distribution, and a WebSocket-first architecture, you'll be able to build your next big thing without worrying about database scaling, configuration, or complex setup.

## Launch Roadmap

### Q2 2025 - Initial Launch
- Core database operations (CRUD)
- Real-time WebSocket connections
- User authentication system
- Basic group management
- Global edge deployment

### Q3 2025 - Advanced Features
- **Lua Cloud Functions** - Deploy serverless edge functions
- **Email Integration** - Send emails directly from your database
- **Third-party Integrations** - One-click Stripe, payment processors, and more
- **Advanced Scaling** - Automatic scaling to millions of users
